# GREAT Lab Mac Studio Connect

Now can use Apple's built-in Screen Sharing feature to log in and establish a private network connection via Tailscale VPN, allowing multiple users to use the Mac Studio simultaneously.

Tip: Tailscale VPN takes some time to establish a connection. After joining or connecting, you need to wait a bit before it works, or you can use the terminal command ping [IP] to check latency and connection status.

## Step 1
Download Tailscale VPN
https://tailscale.com/download

## Step 3
### For MacOS
When I approve it from the backend, use spolight or Launchpad to search "Screen Sharing".
